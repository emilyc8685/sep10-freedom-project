## Content
  The tool I chose was SASS and some things I learned about my tool was SASS varibles. I learned how to use my tools with videos and websites that gave me informtion about how to use SASS. 
  
Sass uses the $ symbol to declare a varible and an example of SASS Variables compared to standard CSS can be:

SASS:
````SASS
  $color1: red;
  $fontsize1: 20px;

p {
   color: $color1;
   font-size: $fontsize1;
}  
````

CSS: 
````CSS
 p {
   color: red;
   font-size: 20px;
}   
````

## Sources

 Some sources that helped me learn more about my tool was with videos. A video i used to learn about @include and @mixin in SASS was this [Sass Crash Course](https://www.youtube.com/watch?v=Zz6eOVaaelI). 

*  @mixin: makes CSS codes reuseable.
  ```` CSS
   @mixin text {
      color: red;
      text-align: center;
      font-size: 20px; 
  }
  ````
* @include: used to include a mixin

```` CSS
     h1 {
      @include text ();
   }
````

 Another way I learned how to use sass was with this website: [SASS Guide](https://www.w3schools.com/sass/default.asp). One thing I learned with the SASS guide was SASS Color Functions.  
 
An example of SASS Color Functions: 

SASS:
````CSS
h2 {
  color: color.complement(#7fffd4)
}
````
CSS: 
````CSS
h2 {
  color: #ff7faa;
}
````
