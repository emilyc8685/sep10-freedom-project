## Tool: **SASS**

### 3/16/26:
* I learned how to use **@include and @mixin** with this [Sass Crash Course](https://www.youtube.com/watch?v=Zz6eOVaaelI).
    *  @mixin: makes CSS codes reuseable.
  ```` CSS
   @mixin text {
      color: red;
      text-align: center;
      font-size: 20px; 
  }
  ````
    * @include: used to include a mixin
   ```` CSS
     h1 {
      @include text ();
   }
   ````
